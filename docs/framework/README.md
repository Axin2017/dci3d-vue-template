# 核心框架
开始使用前，请仔细阅读本文档

## 框架简介
本框架，是以vuejs为基础开发的一款前端框架，集成了常用的工具，组件，做了简单的配置。


## 框架运行

### 安装依赖
```
npm install
```

### 运行调试
```
npm run start
```

### 打包生产模式
```
npm run build
```
